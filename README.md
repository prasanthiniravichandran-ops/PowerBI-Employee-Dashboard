# 📊 HR Analytics Dashboard (Excel Project)

## 🔍 Project Overview

This project is an end-to-end HR Analytics dashboard built in Microsoft Excel to analyze workforce data, employee performance, attrition, and compensation.

The dashboard is designed for **management and client reporting**, providing clear insights into employee trends and business performance.


## 🛠 Tools Used

* Microsoft Excel (Advanced)
* Pivot Tables
* Charts & Visualizations
* Data Cleaning & Transformation

## 📁 Key Metrics Tracked

* Total Workforce: 50
* Active Employees: 40
* Exit Headcount: 10
* Attrition Rate: 20%
* Average Compensation: ₹869,836

## 📊 Dashboard Features

### 👥 Workforce Analysis

* Department-wise employee distribution
* Gender headcount (Male vs Female)
* Employment type distribution (Hybrid, Office, Work From Home)

### 💰 Compensation Insights

* Salary analysis by experience level:

  * Entry
  * Junior
  * Mid
  * Senior

### 📉 Attrition Analysis

* Department-wise attrition distribution
* Top reasons for employee attrition:

  * Career change
  * Commute time
  * Personal reasons
  * Relocation

### 📈 Performance & Recruitment

* Performance appraisal success rate by department
* Recruitment pipeline tracking:

  * Final stage
  * In progress
  * Rejected
  * Shortlisted

### 😊 Employee Insights

* Employee satisfaction score (%)
* Attendance rate by department

## 📌 Key Insights

* Attrition rate is at 20%, indicating scope for retention strategies
* Sales and Finance show strong performance contribution
* Work-from-home and hybrid models form a major portion of employment type
* Employee satisfaction varies across departments, highlighting improvement areas

## 🚀 Business Impact

* Helps HR and management monitor workforce health
* Supports decision-making with data-driven insights
* Improves employee retention strategies
* Provides structured reporting for stakeholders and clients
